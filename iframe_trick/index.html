Iframe 100% Responsive Only With Css

The Very Simple Trick

Many people who need or want to insert an iframe into their site struggle to make this frame fully responsive to all screen sizes! They try to use JavaScript most of the time, which works but can be difficult to manage if you are not familiar with JavaScript! So I will give you a very simple CSS trick! When you see how easy it is to do, you will be surprised !

The Exemple

First look at this example where you have 2 iframes, linked to a html page with one of my pure js animations, they have the same size but inserted in 2 divs with different sizes, resize your browser window, you will see the adaptation of the frames !

Now The Code

CODE HTML5 of the Exemple(with the 2 frames) 

<!DOCTYPE html>
<html>
  <head>
    <title>
      Iframe 100% Trick CSS Exemple
    </title>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css" type="text/css">
  </head>
  <body>
    <div id="site">
      <!-- BIG FRAME HTML CODE START -->
      <div id="bigframe">
        <div class="containerframe">
          <iframe src="frame.html" class="myframe" width="51" height="36">
            </iframe>
        </div>
      </div>
      <!-- BIG FRAME HTML CODE END -->
      <p></p>
      <!-- SMALL FRAME HTML CODE START -->
      <div id="smallframe">
        <div class="containerframe">
          <iframe src="frame.html" class="myframe" width="51" height="36">
            </iframe>
        </div>
      </div>
      <!-- SMALL FRAME HTML CODE END -->
    </div>
  </body>
</html>

CODE CSS for the exemple with The 2 frames ! 

body, html, #site {
  margin: 0;
  padding: 0 ;
}
/* CSS FOR THE EXEMPLE PAGE */
#site {
  width: 100%;/* THE DIV SITE SET TO 100% TO TAKE ALL THE PAGE */
}
#bigframe {
  width: 80%; /* THE BIGFRAME WAS SET TO 80% OF THE DIV SITE  */
  margin: auto;
}
#smallframe {
  width: 40%;/* THE SMALFRAME WAS SET TO 40% OF THE DIV SITE */
  margin: auto;
}
/*END OF CSS FOR THIS PAGE */

/* START OF THE CSS TRICK, IMPORTANT!!! DO NOT CHANGE THE VALUES !!! */
.containerframe {
  position: relative;
  width: 100%;
  height: 0;
  padding-bottom: 56.25%;
}
.myframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
/* END OF THE CSS TRICK */

Like you can see the frame have exactely the same parameter for the 2 size.
Only the size of the div "bigframe" and "smallframe" change in the Css !
With the Css trick the frames adapt to the size of the browser window !

Now The Trick

1: You need to add a class to the iframe ! 

class="myframe"

2: You have to give a ratio to the iframe !
It's just a height and width that you have to give, the values you give don't really matter but you have to put them in ! 

width="51" height="36"

3: The iFrame must be placed in a div ! 

<div class="containerframe">

It looks like this. 

<div class="containerframe">
<iframe src="frame.html" class="myframe" width="51" height="36">
 </iframe>
</div>

The Css Trick

Here is just the Css trick to make the Frame fully Responsive ! you can chage the class Name but
DO NOT CHANGE THE VALUES ! 


/* START OF THE CSS TRICK, IMPORTANT!!! DO NOT CHANGE THE VALUES !!! */
.containerframe {
  position: relative;
  width: 100%;
  height: 0;
  padding-bottom: 56.25%;
}
.myframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
/* END OF THE CSS TRICK */
