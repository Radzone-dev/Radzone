Iframe 100% Responsive Only With Css

The Very Simple Trick

Many people who need or want to insert an iframe into their site struggle to make this frame fully responsive to all screen sizes and maintain the aspect ratio ! They try to use JavaScript most of the time, which works but can be difficult to manage if you are not familiar with JavaScript! So I will give you a very simple CSS trick! When you see how easy it is to do, you will be surprised !

The Exemple

First look at this example where you have 2 iframes, linked to a html page with one of my pure js animations, they are identical but inserted in 2 divs with different sizes, resize your browser window, you will see the adaptation of the iframes !

Like you can see the iframe have exactely the same parameter.
Only the size of the div "bigframe" and "smallframe" change in the Css !
With CSS trick, iframes adapt to the size of the browser window and maintain the aspect ratio !

Now The Trick

1: You need to add a class to the iframe ! 

class="myframe"

2: The iFrame must be placed inside a div ! 

<div class="containerframe">

3: Chose the Aspect Ratio you want for the iframe ! ( Aspect Ratio Exemples )
by changing the padding-top in the Css (16:9, 4:3 ...ect) ! 

padding-top:

The Html looks like this. 

<div class="containerframe">
<iframe src="frame.html" class="myframe">
 </iframe>
</div>

The Css Trick

Here is just the Css trick to make the Frame fully Responsive ! you can chage the class Name but
DO NOT CHANGE ANY VALUES except your desired aspect ratio !
I give you the different Aspect Ratio in the Css ! 


/* START OF THE CSS TRICK, IMPORTANT!!! DO NOT CHANGE ANY VALUES except your desired aspect ratio !!! */
.containerframe {
  position: relative;
  overflow: hidden;
  width: 100%;
  height: 0;
  padding-top: 56.25%;  /* 16:9 Aspect Ratio (divide 9 by 16 = 0.5625) */
/*  padding-top: 75%;        /*   for 4:3 Aspect Ratio */
/*  padding-top: 66.66%;     /*   for 3:2 Aspect Ratio */
/*  padding-top: 177.77%;    /*   for 9:16 Aspect Ratio  phone portrait ratio */
/*  padding-top: 100%;       /*   for 1:1 Aspect Ratio Height and Width is Always Equal */
}
.myframe {
  position: absolute;
  top: 0;
  left: 0;
  bottom: 0;
  right: 0;
  width: 100%;
  height: 100%;
}
/* END OF THE CSS TRICK */
