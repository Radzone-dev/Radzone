Iframe 100% Responsive Only With Css

The Very Simple Trick

Many people who need or want to insert an iframe into their site struggle to make this frame fully responsive to all screen sizes! They try to use JavaScript most of the time, which works but can be difficult to manage if you are not familiar with JavaScript! So I will give you a very simple CSS trick! When you see how easy it is to do, you will be surprised !

The Exemple

First look at this example where you have 2 iframes with the same size but inserted into 2 divs with different sizes,resize the page you see !

Now The Html Code

CODE HTML of the Exemple(with the 2 frames)

<!DOCTYPE html>
<html>
  <head>
    <title>
      Iframe 100% Trick CSS
    </title>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css" type="text/css">
  </head>
  <body>
    <div id="site">
      <!-- BIG FRAME HTML CODE START -->
      <div id="bigframe">
        <div class="containerframe">
          <iframe src="frame.html" class="myframe" width="51.2" height="36">
            </iframe>
        </div>
      </div>
      <!-- BIG FRAME HTML CODE END -->
      <p></p>
      <!-- SMALL FRAME HTML CODE START -->
      <div id="smallframe">
        <div class="containerframe">
          <iframe src="frame.html" class="myframe" width="51.2" height="36">
            </iframe>
        </div>
      </div>
      <!-- SMALL FRAME HTML CODE END -->
    </div>
  </body>
</html>

Like you can see the frame have exactely the same parameter for the 2 size

1: You need to add a class to the iframe ! 
class="myframe"

2: You have to give it a ratio to the iframe (ratio means 'width' and 'height')! 

width="51.2" height="36"
Is just the ratio you want for your frame, in this case I chose to divide 1024 X 720 by 20 (1024÷20=51.2 and 720÷20=36) It's up to you to choose your appropriate ratio and there is no need to divide it.

3:The frames must be enclosed in a div 

<div class="containerframe">

That look like this.

<div class="containerframe">
<iframe src="frame.html" class="myframe" width="51.2" height="36">
 </iframe>
</div>

Now The Css Trick

First the Css for the exemple with The 2 frames !

body, html {
  margin: 0;
  padding: 0 ;
}
/* CSS FOR THIS PAGE */
#site {
  width: 100%;/* THE DIV SITE SET TO 100% TO TAKE ALL THE PAGE */
}
#bigframe {
  width: 80%; /* THE BIGFRAME WAS SET TO 80% OF THE DIV SITE  */
  margin: auto;
}
#smallframe {
  width: 40%;/* THE SMALFRAME WAS SET TO 40% OF THE DIV SITE */
  margin: auto;
}
/*END OF CSS FOR THIS PAGE */

/* START OF THE CSS TRICK, IMPORTANT!!! DO NOT CHANGE THE VALUES !!! */
.containerframe {
  position: relative;
  width: 100%;
  height: 0;
  padding-bottom: 56.25%;
}
.myframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
/* END OF THE CSS TRICK */

Here is just the trick to make the Frame fully Responsive ! you can chage the class Name but
DO NOT CHANGE THE VALUES

Just Add 1 div, Give 1 size, Add 2 class and 8 lines of css, easy no !
