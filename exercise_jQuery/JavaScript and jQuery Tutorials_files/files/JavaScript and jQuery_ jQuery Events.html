on( "click",) Event

The click event is triggered when the mouse button is pressed and released while over the selected element. Example: $("h1").on( "click",). Press and release the mouse button while over an<h1> element to trigger the event. 

$(document).ready(function() {

  $("h1").on( "click",function() {
    $(this).css("background-color","red");
  });

});

on( "mousedown",) and on( "mouseup",) Events

The mousedown event is triggered when the mouse button is pressed while over the selected element. Example: $("h1").on( "mousedown",). Press the mouse button while over an<h1> element to trigger the event.

The mouseup event is triggered when the mouse button is released while over the selected element. Example: $("h1").on( "mouseup",). Release the mouse button while over an<h1> element to trigger the event. 

$(document).ready(function() {

  $("h1").on( "mousedown",function() {
    $(this).css("background-color","red");
  });
  $("h1").on( "mouseup",function() {
    $(this).css("background-color","yellow");
  });

});

Try it .

on( "mouseenter",) and on( "mouseleave",) Events

The mouseenter event is triggered when the mouse cursor enters the selected element. Example: $("h1").on( "mouseenter",). Move the mouse cursor over an<h1> element to trigger the event.

The mouseleave event is triggered when the mouse cursor leaves the selected element. Example: $("h1").on( "mouseleave",). Move the mouse cursor off of an <h1> element to trigger the event. 

$(document).ready(function() {

  $("h1").on( "mouseenter",function() {
    $(this).css("background-color","red");
  });
  $("h1").on( "mouseleave",function() {
    $(this).css("background-color","yellow");
  });

});

Try it .

unbind() Method - Sample 1

The unbind method removes the specified event. Example: $(this).unbind() - All events for the currently triggered event will be removed. 

$(document).ready(function() {

  $("h1").on( "mouseenter",function() {
    $(this).css("background-color","red");
  });
  $("h1").on( "mouseleave",function() {
    $(this).css("background-color","yellow");
    $(this).unbind();
  });

});

Try it .

unbind() Method - Sample 2

The unbind method removes the specified event. Example: $("*").unbind() - All events will be removed. 

$(document).ready(function() {

  $("h1").on( "mouseenter",function() {
    $(this).css("background-color","red");
  });
  $("h1").on( "mouseleave",function() {
    $(this).css("background-color","yellow");
    $("*").unbind();
  });

});

Try it .

unbind() Method - Sample 3

The unbind() method removes the specified event. Example: $("*").unbind("mouseleave") - All mouseleave events will be removed. 

$(document).ready(function() {

  $("h1").on( "mouseenter",function() {
    $(this).css("background-color","red");
  });
  $("h1").on( "mouseleave",function() {
    $(this).css("background-color","yellow");
    $("*").unbind("mouseleave");
  });

});
